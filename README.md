# 加密货币合规情报监控系统

自动收集加密货币合规情报，生成简洁报告，部署到 GitHub Pages。

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    数据收集层                                │
│  ┌──────────────┐  ┌──────────────┐                        │
│  │  X/Twitter   │  │  RSS 新闻源   │                        │
│  │ fetch_x.py   │  │ fetch_rss.py │                        │
│  └──────────────┘  └──────────────┘                        │
└────────────────────┬────────────────────────────────────────┘
                     │
┌────────────────────▼────────────────────────────────────────┐
│                    AI 分析层                                 │
│              analyze.py - K2.5 模型分析                      │
│              分类: P1(紧急) / P2(重要) / P3(一般)            │
└────────────────────┬────────────────────────────────────────┘
                     │
┌────────────────────▼────────────────────────────────────────┐
│                    报告生成层                                │
│              generate.py + Jinja2 模板                       │
│              生成简洁 HTML 报告                              │
└────────────────────┬────────────────────────────────────────┘
                     │
┌────────────────────▼────────────────────────────────────────┐
│                    部署层                                    │
│              GitHub Actions → GitHub Pages                   │
└─────────────────────────────────────────────────────────────┘
```

## 文件结构

```
crypto-compliance/
├── src/
│   ├── config.py          # 配置文件
│   ├── fetch_x.py         # X/Twitter 数据获取
│   ├── fetch_rss.py       # RSS 新闻获取
│   ├── analyze.py         # AI 分析分类
│   └── generate.py        # 报告生成
├── templates/
│   └── report.html        # HTML 模板
├── docs/                  # 生成的报告输出
├── .github/
│   └── workflows/
│       └── deploy.yml     # GitHub Actions 工作流
├── requirements.txt       # Python 依赖
└── README.md
```

## 优先级分类

- **P1 (紧急)**: 监管政策变化、重大执法行动、交易所下架
- **P2 (重要)**: 合规指南更新、行业自律、重要诉讼
- **P3 (一般)**: 行业动态、研究报告、一般新闻

## GitHub Pages 访问地址

https://windhood-jza.github.io/crypto-monitor-pages/

## 本地开发

```bash
# 安装依赖
pip install -r requirements.txt

# 运行完整流程
python src/fetch_x.py
python src/fetch_rss.py
python src/analyze.py
python src/generate.py
```

## 定时任务

系统每天 UTC 00:00 自动运行，生成最新合规情报报告。
